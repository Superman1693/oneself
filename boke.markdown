# 🧩 网站整体架构设计（非常重要）

## 1️⃣ 页面级架构（Page Architecture）

网站应至少包含以下页面（或模块）：

1. 首页（Home）
2. 关于我（About）
3. 项目展示（Projects）
4. 学习笔记 / 博客（Notes / Blog）
5. 设计卡片（Design Cards）
6. 联系方式（Contact / Footer 区域）

> 所有页面应保持 **统一布局逻辑** 与 **统一设计语言**

## 2️⃣ 信息结构设计（Information Architecture）

### 首页（Home）

首页目标不是堆信息，而是 **建立第一印象**：

- 顶部简洁 Hero 区域
  - 姓名 + 一句话定位
  - 一段简短自我介绍
- 下方模块化内容：
  - 精选项目（2–3 个）
  - 最新学习笔记（3 条）
  - 设计卡片预览（横向滑动或网格）
- 明确引导：
  - “查看项目”
  - “阅读笔记”
  - “了解我”

# 🎨 设计风格规范（Design System）

## 整体风格关键词

- 简洁
- 留白
- 高级
- 克制
- 内容优先

## 视觉原则

- 使用中性色（黑 / 白 / 灰）
- 避免高饱和颜色
- 字体清晰、有层级
- 动效轻微、不过度

## 布局原则

- 栅格系统
- 内容居中
- 响应式设计（桌面优先，兼容移动端）
